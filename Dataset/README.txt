******1. Introductory information******

------Data Name:------

Theoretical test for the work "Hinterland freight transportation replanning model under the framework of synchromodality" 


It is the theoretical dataset to support the replanning model from the publication 'Hinterland freight transportation replanning model under the framework 
of synchromodality' published on Transportation Research Part E. (pending acceptance). It is made up of three parts: test cases with different network sizes,
those of diffrent shipment amount, and those of different network topologies. 

------Naming convention:------

X_ ntw_Y_nodes_Z shipment: 

X: the network topology; Y: the number of nodes in the tested network. Z the number of shipments in the tested network.  
In the case of X defaulted missing, it is partially connected mesh topology. 


The first two groups (test cases of different network size and different amount of shipments) are the same topology with the Rotterdam hinterland 
transportation network, i.e., partially connected mesh topology. Each excel is named by the network size, and the number of shipments. For the third groups, 
test cases of different topologies, each one is named by the topology of the network, network size and the number of the 
shipments of the testcases. For more information about network topology, please check https://en.wikipedia.org/wiki/Network_topology. 


------Format of the formation:------
Each excel contains two sheets. Sheet 1 of each one is attached with a figure of the network. Sheet 1 also contains the information
of terminals, the services, the transhipments operations and the shipments. The definition and usage of these data can be seen from the manuscript of the
above mentioned publication. Sheet 2 describe the replanning results of the model, including the shipment flow assignment and the operating time of the 
services, and the corresponding costs.

Contact information: for any questions please contact Wenhua Qu via quwenhualiz@gmail.com. 


******2. Methodological information******
As mentioned, the dataset is synthetic, to test the applicability of the replanning model presented from the publication. The theoretical data is generated
 based on the real data from the Rotterdam hinterland freight transportation network. The description of the real data can be checked from the publication. 

The unit of time in the data set is the hour, the unit of cost is the euro.
